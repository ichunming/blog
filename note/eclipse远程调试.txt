2. jetty远程调试
一.Run>External Tools>External Tools Configuration>Program-->New
Location 中设置mvn的执行路径（比如/JavaDev/apache-maven-3.0.3/bin/mvn）
Working Directory 中选择webapp所属的workspace
在“Arguments”中添加 jetty:run
Environment添加变量MAVEN_OPTS=-Xdebug -Xnoagent -Djava.compiler=NONE -Xrunjdwp:transport=dt_socket,address=8000,server=y,suspend=y
如果设置suspend=n，则运行时不需要等待debugger
二.Run>Debug>Debug Configuration>Remote Java Application-->New
Project 中选择webapp项目。并确保端口号与步骤一中 address= 设置的值相同。
关联源码：选择 Source 标签，选择 Add… 按钮添加关联的源代码
 运行一.Run>External Tools>Program
 运行二.Run/Debug”中选择步骤二中创建的debugger，连接上后，即可以进行debug
